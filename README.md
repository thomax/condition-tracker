# R.E.S.I.C.O.T.

Rediculously Simple Condition Tracker - for various TTRPGs. Beginning with SWADE and 5e.

## Target ability syntax

### Traits
`traits.*` all attribute and skill rolls

`traits.attributes.*` all attribute rolls 

`traits.attributes.spirit` spirit rolls 

`traits.attributes.smarts|spirit` smarts and spirit rolls 

`traits.skills.*` all skill rolls

`traits.skills.common-knowledge` common knowledge rolls

`traits.skills.stealth|thievery|driving` stealth, thievery and driving rolls

### Derived stats
`derived.pace` pace (movement speed)

`derived.run` run die

`derived.parry` parry

`derived.size` 

`derived.toughness` 

### Other

`other.wounds`

`other.fatigue`

`other.bennies`

`other.power`
